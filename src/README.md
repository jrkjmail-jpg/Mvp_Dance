# Будущая структура frontend-кода

Папка `src` создана как каркас для постепенного упорядочивания проекта.

Пока рабочий код остается в корневых файлах:

- `index.html`
- `app.js`
- `styles.css`
- `sw.js`

Не переносить код сюда без отдельной проверки и отдельного коммита.

Рекомендуемый порядок переноса:

1. `storage`
2. `scoring`
3. `pose`
4. `media`
5. `ui`
6. `student`
7. `teacher`
8. `studio`
