# Будущая структура CSS

Папка создана как каркас для постепенного разделения большого `styles.css`.

Пока рабочие стили остаются в корневом `styles.css`.

Переносить CSS сюда нужно аккуратно и только с визуальной проверкой.

Рекомендуемая будущая структура:

```text
styles/
  tokens.css
  base.css
  layout.css
  components/
  screens/
  responsive.css
```
