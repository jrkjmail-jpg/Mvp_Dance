# Танцуй

Прототип российского приложения для домашнего повторения танцев в игровом формате.

Полная структура, функционал и техническое задание для переноса находятся в
[`docs/full-migration-spec.md`](docs/full-migration-spec.md).

## Запуск

```bash
python3 -m http.server 4173
```

Открыть:

```text
http://127.0.0.1:4173
```

## Что уже есть

- кабинет педагога и кабинет ученика;
- mobile-first интерфейс для телефона;
- единый универсальный дизайн для всех ролей;
- лента выбора танцев;
- раздельная загрузка двух видео педагога: обучающего разбора и танца под музыку;
- автоматическое считывание эталонной позы только из видео под музыку;
- экран прогресса во время считывания эталона без мелькания скелета;
- проверка эталонного скелета педагогом поверх видео;
- отдельные маркеры начала и конца для обучающего видео и видео под музыку;
- переключатель видео педагога `Учить / Сдать` в рабочем плеере;
- переключатели видимости видео и скелета в кабинете педагога;
- камера ученика;
- режим обучения с крупным видео педагога на весь экран;
- сдача с TikTok-подобным отсчётом перед стартом записи;
- зеркальный режим;
- игровой режим со звездами и уровнями;
- экран результата после сдачи с большой звездной оценкой;
- длинная шкала из 30 уровней с названиями;
- сравнение углов и нормализованной формы тела;
- экзамен как запись под музыкальную дорожку педагога;
- сравнение рук, ног, корпуса, грудной клетки, головы, пальцев и тайминга;
- штраф за пропущенные моменты и рассинхрон с музыкой;
- сохранение последних попыток в `localStorage`.

## Ограничения прототипа

- MediaPipe загружается через CDN, поэтому нужен интернет при первом запуске;
- точность зависит от света, полного роста в кадре и неподвижной камеры;
- пальцы пока считаются по точкам `thumb/index/pinky` из Pose Landmarker, для полноценной кисти нужен отдельный Hand Landmarker;
- видео ученика не сохраняется, сохраняются только попытки и баллы в браузере.
